# Frontend - Ekşi Sözlük Benzeri Platform

Next.js 14 ile geliştirilmiş React frontend.

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
# Development
npm run dev

# Production
npm run build
npm run start
```

## Yapılandırma

`.env.local` dosyasında API URL'ini ayarlayabilirsiniz:

```env
NEXT_PUBLIC_API_URL=http://localhost:3001
```

Frontend `http://localhost:3000` adresinde çalışır.
